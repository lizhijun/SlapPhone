# SlapMac 软件功能深度分析

> 官网: https://slapmac.com/

## 软件概述

SlapMac 是一款运行在 macOS 菜单栏的创意娱乐应用，核心玩法：**拍打 MacBook，它会发出声音回应**。通过内置传感器检测物理拍打，根据力度自适应音量反馈，是一款兼具创意与趣味的桌面应用。

---

## 核心功能

### 1. 拍打检测

- 通过 MacBook 内置传感器检测拍打力度
- 力度自适应音量：轻拍 = 低声呢喃，重拍 = 全音量尖叫
- 菜单栏静默运行，无 Dock 图标
- 不会在正常打字时误触

### 2. USB Moaner 模式

- 独立功能模式
- 每次插入/拔出 USB 设备时自动触发声音反应
- 无需拍打，自动响应

### 3. 声音包系统

- **8 种情绪/风格声音包**
- **130+ 音频片段库**
- 包含多种特色风格
- 支持组合击打音效（v1.0.15+）

### 4. 可调参数

| 参数       | 说明                                     |
| ---------- | ---------------------------------------- |
| 灵敏度     | 从"地震检测器"级别到"需要助跑"级别，自由调节 |
| 冷却时间   | 默认 1.0 秒，防止快速连续拍打导致系统过载   |
| 评论员语音 | 可选的评论员配音，支持开关切换               |

### 5. 其他功能

- **开机自启** - 应用启动时自动运行
- **拍打计数器** - 菜单栏显示终身拍打次数
- **屏幕闪光特效** - 拍打时屏幕闪光（v1.0.15+）
- **新手引导** - 首次使用的友好欢迎界面

---

## 技术规格

### 系统要求

- **macOS 14.6+**（Sonoma 及以上）
- **仅支持 Apple Silicon（M1+ 芯片 MacBook）**
- 不支持 Intel Mac

### 性能特点

- 不影响电池续航
- 不会在正常打字时误触
- 菜单栏常驻，资源占用极小
